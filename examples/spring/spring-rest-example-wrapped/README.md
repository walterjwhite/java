2. show use of exception advice to log exception
3. capture web traffic
4. export web traffic to CSV
5. secure endpoints with basic auth using Spring Security
6. filter traffic by CIDR
7. react UI
8. async
9. simple batch jobs without using Spring Batch
10. export transactional data to CSV
11. Spring MVC with Thymeleaf and JPA
12. rate-limiting with Resilience4j and Byte Buddy
